# @cagent/ui

Shared React UI components for [Cagent](https://github.com/kashyaprpuranik/cagent) frontends.

## Components

Badge, BandwidthWidget, BlockedDomainsWidget, BlockedTimeseriesChart, Button, Card, DiagnoseModal, Input, Modal, Select, Table, ToastContainer, useToast.

## Installation

```json
{
  "dependencies": {
    "@cagent/ui": "github:kashyaprpuranik/cagent-ui"
  }
}
```

## Peer Dependencies

- React 18+ (react, react-dom)
- lucide-react

## Usage

```tsx
import { Button, Card, Table } from '@cagent/ui';
```

Includes a Tailwind CSS preset (`tailwind.preset.js`) for consistent styling.
